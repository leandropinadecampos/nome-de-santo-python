# Nome de Santo

Programa em Python que verifica se um nome digitado é nome de santo.

## Como funciona

O usuário digita seu nome e o programa analisa se o primeiro nome está na lista de santos.

## Exemplo de uso

Digite seu nome:

Leandro

Saída do programa:

Parabéns! O nome Leandro é nome de santo!

## Tecnologias usadas

- Python
